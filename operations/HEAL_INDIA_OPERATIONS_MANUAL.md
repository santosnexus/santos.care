## 1. EXECUTIVE OVERVIEW

### 1.1 Purpose of This Manual

This Operations Manual provides comprehensive Standard Operating Procedures (SOPs) for the management of international medical tourists arriving in India through Heal India Medi Tourism (santos.care). It serves as:

- **Training Document** for new staff members
- **Reference Guide** for existing team members
- **Quality Standard** for service delivery expectations
- **Stakeholder Communication** tool for partners and investors
- **Compliance Framework** for regulatory requirements

### 1.2 Service Promise

> "From first inquiry to full recovery, we walk with you every step of the way."

**Core Service Pillars:**
1. Transparency — No hidden costs, clear communication at every stage
2. Speed — 24-hour response to inquiries, priority hospital appointments
3. Safety — JCI/NABH accredited hospital network, qualified practitioners
4. Comfort — End-to-end logistics, cultural sensitivity, personalized care
5. Continuity — Post-treatment follow-up, telemedicine support

### 1.3 Operational Boundaries

| IN SCOPE | OUT OF SCOPE |
|----------|-------------|
| Patient inquiry handling | Medical diagnosis (done by hospital doctors) |
| Hospital/doctor appointment scheduling | Surgical procedures (hospital responsibility) |
| Visa assistance documentation | Billing disputes with hospitals |
| Travel booking (flights, transport) | Medical complications (hospital's liability) |
| Accommodation arrangements | Insurance claims processing (we assist only) |
| Airport transfers | Cosmetic procedures without medical need |
| Language interpretation | Experimental treatments |
| Post-treatment Ayurveda recovery | Visa overstay issues |
| Telemedicine follow-up (30 days) | Long-term medication sourcing |

### 2.2 Decision Gates

| Gate | Criteria to Proceed | Escalation if Not Met |
|------|---------------------|-----------------------|
| **GATE 1: Inquiry → Qualification** | Medical reports reviewed, patient contact confirmed | Re-engage with alternate contact method |
| **GATE 2: Qualification → Treatment Plan** | Clear treatment need identified, budget discussed | Send additional questions, offer second opinion review |
| **GATE 3: Treatment Plan → Confirmation** | Patient agrees to hospital/treatment, deposit received | Address concerns, offer alternatives |
| **GATE 4: Confirmation → Travel** | Valid visa (or visa on arrival), flights booked | Assist visa issues, provide alternatives |
| **GATE 5: Travel → Arrival** | Flight details confirmed, accommodation verified | Re-book if issues, provide alternative |
| **GATE 6: Arrival → Treatment** | Patient arrived safely, hospital admission confirmed | Arrange emergency consultation if delayed |
| **GATE 7: Treatment → Discharge** | Treatment complete, patient stable, fitness confirmed |延 Extend hospital stay if needed |
| **GATE 8: Discharge → Recovery** | Transfer to Ayurveda confirmed, patient comfortable | Arrange alternative if unavailable |
| **GATE 9: Recovery → Return** | Patient cleared to travel, return flights booked |延延 medical clearance if concerns |
| **GATE 10: Return → Follow-up** | Patient arrived home, first follow-up scheduled | Immediate call if unreachable |
