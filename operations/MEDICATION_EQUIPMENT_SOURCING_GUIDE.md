## 1. MEDICATION SOURCING OVERVIEW

### 1.1 When Patients Need Medication

```
SCENARIOS REQUIRING MEDICATION SOURCING:

1. POST-DISCHARGE MEDICATIONS
   • Pain management
   • Antibiotics
   • Blood thinners
   • BP/heart medications
   • Diabetes medications

2. ONGOING MEDICATIONS FOR RETURN
   • MS patients (免疫调节剂)
   • Cancer patients (靶向药物)
   • Chronic conditions

3. AYURVEDA MEDICATIONS
   • Herbal formulations
   • Panchakarma medicines
   • GMP-certified products

4. SPECIAL MEDICATIONS
   • refrigerated items (insulin, biologics)
   • Controlled substances (strict regulations)
```

### 1.2 Sourcing Options

| Source | Use Case | Pros | Cons |
|--------|----------|------|------|
| **Hospital Pharmacy** | During admission | Accurate, prescribed | May be expensive |
| **Local Pharmacy** | Post-discharge | Cost savings, generic available | Requires prescription |
| **Online Pharmacy** | Bulk/ongoing | Convenience, home delivery | Delivery time |
| **International Shipping** | For patients abroad | No travel needed | 7-10 days, regulated |

## 3. MEDICATION PROCUREMENT PROTOCOL

### 3.1 Discharge Medication Checklist

```
DURING HOSPITAL DISCHARGE:

□ Doctor writes discharge prescription
□ Pharmacist prepares medications
□ Verify all medications are included:
   • Names of all medications
   • Dosage (morning/afternoon/evening/night)
   • Duration (how many days/weeks)
   • With food or empty stomach
   • Storage requirements (refrigerate?)
□ Understand purpose of each medication
□ Note side effects to watch for
□ Get prescription copy for records
□ Ask pharmacist about refills

COST-SAVING OPTIONS:

Option A: Hospital pharmacy
• Convenient
• Exact medications
• May be 20-30% more expensive

Option B: External pharmacy
• Show discharge prescription
• Request generic alternatives
• Cost 30-50% savings
• Coordinate with coordinator

Option C: Bulk purchase
• For ongoing medications
• Ask for full course discount
• Combine with other patients (if same meds)
```

### 3.2 International Medication Shipping

```
FOR PATIENTS RETURNING ABROAD:

PROTOCOL FOR SHIPPING MEDICATIONS:

Step 1: Doctor prescribes medications
Step 2: Confirm which can be shipped internationally
Step 3: Obtain:
   • Valid prescription (doctor's letterhead)
   • Patient details (name, diagnosis, address)
   • Medication list (names, quantities, values)
Step 4: Pack properly (temperature-sensitive items need cooling)
Step 5: Ship via international courier (7-10 days)
Step 6: Patient receives at home country address

REGULATED ITEMS (CANNOT SHIP):
• Controlled substances (morphine, codeine, etc.)
• Some psychotropic medications
•注射药品 (injections requiring cold chain)
• Anything restricted by destination country

ALLOWED ITEMS:
• Oral medications in original packaging
• Prescription medications with valid Rx
• Over-the-counter items (basic)
• Ayurveda herbal medicines (with documentation)

COST ESTIMATE:
• International shipping: $30-80 (depending on weight)
• Customs duties may apply in destination country
• Recommend patient declares at customs

PROVIDERS:
• DHL International: +91 1800 111 233
• FedEx: +91 1800 266 234
• Aramex: +91 1800 100 3636
```

### 5.2 Equipment Selection Guide

```
WHICH EQUIPMENT FOR WHICH PATIENT:

POST-ORTHOPEDIC SURGERY:
✓ Crutches or walker (patient mobility)
✓ Wheelchair (for long distances)
✓ Hospital bed (if needed for comfort)
✓ Blood pressure monitor (if on BP meds)

POST-CARDIAC SURGERY:
✓ Pulse oximeter (oxygen monitoring)
✓ Blood pressure monitor
✓ Weighing scale (fluid retention)
✓ Oxygen concentrator (if prescribed)

MS PATIENTS:
✓ Wheelchair (if mobility issues)
✓ Walker (if unsteady gait)
✓ Hospital bed (if spending time resting)
✓ Nebulizer (if respiratory issues)

CANCER PATIENTS:
✓ Hospital bed (if weak, fatigued)
✓ Wheelchair (for mobility)
✓ Oxygen concentrator (if lung involvement)

ONGOING CARE:
✓ BP monitor (cardiac patients)
✓ Glucose monitor (diabetic patients)
✓ Pulse oximeter (respiratory patients)
```

## 7. SPECIAL CONSIDERATIONS

### 7.1 Temperature-Sensitive Medications

```
REFRIGERATION REQUIREMENTS:

COMMON ITEMS:
• Insulin
• Some antibiotics (眼药水)
• Biologics
• Certain injections
• Some Ayurveda medicines

HANDLING:
□ Use insulated bags/coolers during transport
□ For travel: Use gel packs (not directly on medication)
□ At home: Ensure refrigerator temp 2-8°C
□ For international shipping: Requires cold chain packaging

COORDINATOR ACTION:
□ Note refrigeration requirement
□ Inform patient
□ Provide cooling bag for transport
□ For shipping: Use specialized courier with cold chain
```

### 7.2 Controlled Substances

```
REGULATED MEDICATIONS:

In India, certain medications require special prescriptions:
• Some pain medications (morphine, codeine)
• Some sedatives
• Some psychotropic medications

FOR PATIENTS:
□ Understand which medications are controlled
□ Hospital pharmacy can provide limited quantities
□ For international travel: Check destination country regulations
□ Cannot ship controlled substances internationally without special permits

RECOMMENDATION:
□ Use alternatives when possible
□ Hospital pharmacy manages within legal limits
□ For extended needs: Local doctor in home country manages refills
```

## 8. EMERGENCY MEDICATION PROTOCOL

```
IF PATIENT RUNS OUT OF MEDICATION ABROAD:

IMMEDIATE STEPS:
1. Contact coordinator via WhatsApp
2. Doctor's prescription needed (photo of original)
3. Identify if medication available locally in home country
4. If not available locally, ship urgently

URGENT SHIPPING:
□ Express international courier (3-5 days)
□ Document: Prescription + patient letter
□ Declaration for customs

PREVENTION:
□ Calculate medication needs for entire trip + buffer
□ Extra medication for extended stays
□ Prescription copy in luggage and with attendant
□ List of generic names (in case brand differs)
```
